📖Hướng Dẫn Setup Google Sheets

Bước 1 — Tạo Google Sheet

Mở Google Sheets, tạo file mới tên "HR System"

Bước 2 — Tạo Apps Script

Trong Sheet: Extensions → Apps Script → paste code từ file google-apps-script.js

Bước 3 — Deploy

Deploy → New Deployment → Web App → Execute as Me → Anyone → Copy URL

Bước 4 — Điền URL

Paste URL vào ô "Google Apps Script URL" ở trên → Lưu
